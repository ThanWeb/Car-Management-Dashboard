## Endpoints

- /                   : Home Page
- /cars               : List Car Page
- /cars/add           : Create New Car Page
- /cars/edit/:id      : Edit Car Page

- /cars               : API Endpoint for Create New Car
- /cars/edit/:id      : API Endpoint for Edit Car
- /cars/delete/:id    : API Endpoint for Delete Car
